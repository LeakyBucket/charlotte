Charlotte v0.0.1

Modules summary

Charlotte

Charlotte is an Elixir Web Framework. It is inspired by both Rails and Sinatra.

Charlotte.Controller
Charlotte.Dispatcher

The functions in Charlotte.Dispatcher are intended for building a proper structure for Cowboy's routing.

Charlotte.Handlers.HTTP

The HTTP Handler module provides the basic HTTP 1.1 callbacks Cowboy expects in a route handler.

Charlotte.Logger
Charlotte.Req

The Charlotte Request Handler contains functions for processing the Cowboy request object.
Charlotte.Supervisor
Charlotte.Views.Compiler

The Views Compiler builds view modules for all the view files found at the specified path.

Charlotte.Webserver

The Webserver handles the underlying cowboy process.
